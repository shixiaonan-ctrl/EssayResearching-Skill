# MLA 排版格式规范

> 适用于现代语言协会（MLA）规范的人文学科、语言文学领域学术论文。

## 页面设置

| 参数 | 值 |
|------|-----|
| 纸张 | US Letter (8.5"×11") |
| 边距 | 四周均为 1 inch |
| 分栏 | 单栏 |

## 标题与层级

| 元素 | 字体 | 字号 | 样式 | 对齐 |
|------|------|------|------|------|
| 第一页左上角信息 | Times New Roman | 12pt | (作者、导师、课程、日期) 多行排版 | 左对齐 |
| 论文标题 | Times New Roman | 12pt | 正常（无加粗或斜体） | 居中 |
| 一级标题 | Times New Roman | 12pt | 加粗 | 左对齐，或居中 |
| 二级标题 | Times New Roman | 12pt | 斜体 | 左对齐 |

## 正文

| 参数 | 值 |
|------|-----|
| 字体 | Times New Roman |
| 字号 | 12pt |
| 对齐 | 左对齐（不对齐右边界） |
| 首行缩进 | 0.5 inch |
| 行距 | 双倍行距 (Double spacing) |
| 段前段后间距 | 0（通过双倍行距区分） |

## 引用与长引文

| 参数 | 值 |
|------|-----|
| 超过4行的引用 | Times New Roman |
| 缩进 | 整体向左缩进 0.5 inch |
| 引号 | 无需加引号 |

## 图表注

| 参数 | 值 |
|------|-----|
| 位置 | 图下方，表上方或下方 |
| 前缀 | "Fig. 1." 或 "Table 1." |

## 参考文献 (Works Cited)

| 参数 | 值 |
|------|-----|
| 标题 | "Works Cited" (居中) |
| 字号 | 12pt |
| 缩进 | 悬挂缩进 0.5 inch |
| 行距 | 双倍行距 |
| 排序 | 字母顺序 |

## 页眉页脚

| 参数 | 值 |
|------|-----|
| 页码 | 页面右上角，距离顶边 0.5 inch |
| 格式 | [作者姓氏] [页码] (e.g., "Smith 1") |
