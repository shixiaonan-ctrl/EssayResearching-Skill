# 中国学位论文排版格式规范（通用参考）

> 适用于国内高校本科/硕士/博士学位论文。具体参数以各学校要求为准，本文件为通用默认值。由主 SKILL.md 的格式解析步骤（§2.2）读取，写入 `config.json` 的 `format_params`。

## 页面设置

| 参数 | 值 |
|------|-----|
| 纸张 | A4 (210mm×297mm) |
| 上边距 | 3cm |
| 下边距 | 2.5cm |
| 左边距 | 3cm |
| 右边距 | 2.5cm |
| 分栏 | 单栏 |

## 标题与层级

| 层级 | 字体 | 字号 | 样式 | 对齐 |
|------|------|------|------|------|
| 论文题目 | 黑体 | 三号 (16pt) | 加粗 | 居中 |
| 一级标题（章） | 黑体 | 小三号 (15pt) | 加粗，格式"第X章 标题" | 居中 |
| 二级标题（节） | 黑体 | 四号 (14pt) | 加粗，格式"X.X 标题" | 左对齐 |
| 三级标题 | 黑体 | 小四号 (12pt) | 加粗，格式"X.X.X 标题" | 左对齐 |
| 四级标题 | 宋体 | 小四号 (12pt) | 加粗 | 左对齐 |

## 正文

| 参数 | 值 |
|------|-----|
| 中文字体 | 宋体 |
| 英文字体 | Times New Roman |
| 字号 | 小四号 (12pt) |
| 对齐 | 两端对齐 |
| 首行缩进 | 2 字符（约 24pt） |
| 行距 | 1.5 倍行距 |

## 摘要

| 参数 | 值 |
|------|-----|
| 中文摘要标题 | 黑体 小三号居中，"摘 要"（中间空一格） |
| 中文摘要正文 | 宋体 小四号 |
| 英文摘要标题 | Times New Roman 小三号居中加粗，"Abstract" |
| 英文摘要正文 | Times New Roman 小四号 |
| 关键词 | "关键词：" 黑体，内容宋体，分号分隔 |

## 图注

| 参数 | 值 |
|------|-----|
| 字体 | 宋体 |
| 字号 | 五号 (10.5pt) |
| 位置 | 图下方，居中 |
| 格式 | "图 X-Y 说明文字"（X=章号，Y=图序号） |

## 表注

| 参数 | 值 |
|------|-----|
| 字体 | 宋体 |
| 字号 | 五号 (10.5pt) |
| 位置 | 表上方，居中 |
| 格式 | "表 X-Y 说明文字" |

## 公式

| 参数 | 值 |
|------|-----|
| 格式 | 原生 OMML |
| 位置 | 居中 |
| 编号 | 靠右对齐，格式 `(X-Y)`（X=章号，Y=公式序号） |

## 参考文献

| 参数 | 值 |
|------|-----|
| 字体 | 宋体 |
| 字号 | 五号 (10.5pt) |
| 格式 | GB/T 7714 |
| 标题 | 黑体小三号居中，"参考文献" |

## 页眉页脚

| 参数 | 值 |
|------|-----|
| 页码 | 底部居中，阿拉伯数字 |
| 页眉 | 居中，五号宋体，内容为学校名称或论文标题 |
| 页眉线 | 单线 |
| 首页 | 封面页无页眉页码 |

## 特殊说明

> [!NOTE]
> 各学校的学位论文格式细节差异较大（如章节编号方式、封面格式、致谢位置等）。如果用户提供了学校的 `.docx` 模板，**模板优先**，本文件作为兜底默认值。
