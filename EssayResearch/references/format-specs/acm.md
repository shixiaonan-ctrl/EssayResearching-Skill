# ACM 排版格式规范

> 适用于 ACM 会议论文和期刊论文（如 SIGGRAPH, CHI 等）。由主 SKILL.md 的格式解析步骤（§2.2）读取，写入 `config.json` 的 `format_params`。

## 页面设置

| 参数 | 值 |
|------|-----|
| 纸张 | US Letter (8.5"×11") |
| 上边距 | 0.75 inch |
| 下边距 | 1 inch |
| 左边距 | 0.75 inch |
| 右边距 | 0.75 inch |
| 分栏 | 双栏（标准会议模板，摘要/标题单栏） |
| 栏间距 | 0.33 inch |

## 标题与层级

| 元素 | 字体 | 字号 | 样式 | 对齐 |
|------|------|------|------|------|
| 论文标题 | Libertine / Times | 18pt | 加粗 | 居中 |
| 作者 | Libertine / Times | 11pt | 正常 | 居中 |
| 机构/邮箱 | Libertine / Times | 10pt | 正常 | 居中 |
| 一级标题 | Libertine / Times | 11pt | 加粗，全大写 (`1 INTRODUCTION`) | 左对齐 |
| 二级标题 | Libertine / Times | 10pt | 加粗，首字母大写 (`1.1 Background`) | 左对齐 |
| 三级标题 | Libertine / Times | 10pt | 斜体，首字母大写 | 左对齐，缩进 |

## 正文

| 参数 | 值 |
|------|-----|
| 字体 | Libertine / Times New Roman |
| 字号 | 9pt |
| 对齐 | 两端对齐 |
| 首行缩进 | 0.15 inch |
| 行距 | 单倍行距 |
| 段前间距 | 0 |
| 段后间距 | 0 |

## 摘要

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 样式 | 正常 |
| 缩进 | 左右无额外缩进（由分栏决定） |
| 开头 | "ABSTRACT" 一级标题格式 |

## 公式

| 参数 | 值 |
|------|-----|
| 格式 | 原生 OMML |
| 位置 | 居中 |
| 编号 | 靠右对齐，格式 `(1)` |

## 图注

| 参数 | 值 |
|------|-----|
| 字号 | 8pt |
| 位置 | 图下方，居中 |
| 前缀 | "Figure N:" 正常（不加粗） |
| 间距 | 图注与正文间 10pt |

## 表注

| 参数 | 值 |
|------|-----|
| 字号 | 8pt |
| 位置 | 表上方，居中 |
| 前缀 | "Table N:" |

## 参考文献

| 参数 | 值 |
|------|-----|
| 字号 | 8pt |
| 缩进 | 悬挂缩进 0.15 inch |
| 编号 | `[1]`, `[2]`, ... |

## 页眉页脚

| 参数 | 值 |
|------|-----|
| 页码 | 底部居中（或不显示，根据具体模板要求） |
| 页眉 | ACM会议名称缩写等信息的页眉（左/右对齐） |
