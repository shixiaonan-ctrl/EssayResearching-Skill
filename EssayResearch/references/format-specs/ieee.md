# IEEE 排版格式规范

> 适用于 IEEE 会议论文和期刊论文。由主 SKILL.md 的格式解析步骤（§2.2）读取，写入 `config.json` 的 `format_params`。

## 页面设置

| 参数 | 值 |
|------|-----|
| 纸张 | US Letter (8.5"×11") |
| 上边距 | 1 inch (1440 DXA) |
| 下边距 | 1 inch |
| 左边距 | 1 inch |
| 右边距 | 1 inch |
| 分栏 | 双栏（标题/作者/摘要区域单栏） |
| 栏间距 | 0.25 inch (360 DXA) |

## 标题与层级

| 元素 | 字体 | 字号 | 样式 | 对齐 |
|------|------|------|------|------|
| 论文标题 | Times New Roman | 24pt | 加粗 | 居中 |
| 作者 | Times New Roman | 11pt | 正常 | 居中 |
| 机构/邮箱 | Times New Roman | 10pt | 正常 | 居中 |
| 一级标题 | Times New Roman | 10pt | 加粗，罗马数字 + 全大写 (`I. INTRODUCTION`) | 居中 |
| 二级标题 | Times New Roman | 10pt | 斜体，字母编号 (`A. Background`) | 左对齐 |
| 三级标题 | Times New Roman | 10pt | 斜体，阿拉伯数字编号 (`1) Details`) | 左对齐，缩进 |

## 正文

| 参数 | 值 |
|------|-----|
| 字体 | Times New Roman |
| 字号 | 10pt |
| 对齐 | 两端对齐 |
| 首行缩进 | 0.25 inch |
| 行距 | 单倍行距 |
| 段前间距 | 0 |
| 段后间距 | 0 |

## 摘要

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 样式 | 斜体 |
| 缩进 | 左右各 0.5 inch |
| 开头 | "Abstract—" 加粗（非斜体） |

## 公式

| 参数 | 值 |
|------|-----|
| 格式 | 原生 OMML |
| 位置 | 居中 |
| 编号 | 靠右对齐，格式 `(1)` |

## 图注

| 参数 | 值 |
|------|-----|
| 字号 | 8pt |
| 位置 | 图下方，居中 |
| 前缀 | "Fig. N." 加粗 |
| 间距 | 图与图注间 6pt，图注与正文间 12pt |

## 表注

| 参数 | 值 |
|------|-----|
| 字号 | 8pt |
| 位置 | 表上方，居中 |
| 前缀 | "TABLE N" 全大写加粗 |

## 参考文献

| 参数 | 值 |
|------|-----|
| 字号 | 8pt |
| 缩进 | 悬挂缩进 0.25 inch |
| 编号 | `[1]`, `[2]`, ... |

## 页眉页脚

| 参数 | 值 |
|------|-----|
| 页码 | 不显示（会议版本）；底部居中（期刊版本） |
| 页眉 | 无 |
