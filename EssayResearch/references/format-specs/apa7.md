# APA 第7版排版格式规范

> 适用于心理学、社会科学等领域论文。由主 SKILL.md 的格式解析步骤（§2.2）读取，写入 `config.json` 的 `format_params`。

## 页面设置

| 参数 | 值 |
|------|-----|
| 纸张 | US Letter (8.5"×11") |
| 上边距 | 1 inch |
| 下边距 | 1 inch |
| 左边距 | 1 inch |
| 右边距 | 1 inch |
| 分栏 | 单栏 |

## 标题与层级

| 层级 | 字体 | 字号 | 样式 | 对齐 |
|------|------|------|------|------|
| 论文标题 | Times New Roman | 12pt | 加粗，Title Case | 居中 |
| 一级标题 | Times New Roman | 12pt | 加粗 | 居中 |
| 二级标题 | Times New Roman | 12pt | 加粗 | 左对齐 |
| 三级标题 | Times New Roman | 12pt | 加粗斜体 | 左对齐 |
| 四级标题 | Times New Roman | 12pt | 加粗，句末加句号 | 首行缩进 0.5 inch |
| 五级标题 | Times New Roman | 12pt | 加粗斜体，句末加句号 | 首行缩进 0.5 inch |

## 正文

| 参数 | 值 |
|------|-----|
| 字体 | Times New Roman |
| 字号 | 12pt |
| 对齐 | 左对齐（APA 推荐）或两端对齐 |
| 首行缩进 | 0.5 inch |
| 行距 | 双倍行距 |
| 段前间距 | 0 |
| 段后间距 | 0 |

## 摘要

| 参数 | 值 |
|------|-----|
| 字号 | 12pt |
| 样式 | 正常（非斜体） |
| 标题 | "Abstract" 居中加粗 |
| 首行缩进 | 无（摘要段落不缩进） |
| 关键词 | 摘要后另起一段，"Keywords:" 斜体，后跟关键词 |

## 图注

| 参数 | 值 |
|------|-----|
| 序号行 | "Figure N" 加粗斜体，图上方或图下方 |
| 标题行 | 图标题，斜体，紧跟序号行 |
| 说明 | 正常字体，换行后写 |

## 表注

| 参数 | 值 |
|------|-----|
| 序号行 | "Table N" 加粗斜体，表上方 |
| 标题行 | 表标题，斜体 |

## 参考文献

| 参数 | 值 |
|------|-----|
| 字号 | 12pt |
| 行距 | 双倍行距 |
| 缩进 | 悬挂缩进 0.5 inch |
| 排序 | 按作者姓氏字母排序 |
| 标题 | "References" 居中加粗 |

## 页眉页脚

| 参数 | 值 |
|------|-----|
| 页码 | 右上角，阿拉伯数字 |
| 运行页眉 | 左上角，全大写，≤50 字符（论文标题缩写） |
| 首页 | 与其他页相同 |
