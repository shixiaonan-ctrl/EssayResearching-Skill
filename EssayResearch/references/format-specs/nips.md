# NeurIPS 排版格式规范

> 适用于 NeurIPS (Neural Information Processing Systems) 机器学习相关学术会议论文。

## 页面设置

| 参数 | 值 |
|------|-----|
| 纸张 | US Letter (8.5"×11") |
| 内容宽度 | 5.5 inch (左右边距通常为 1.5 inch) |
| 上下边距 | 约 1.5 inch |
| 分栏 | 单栏 |

## 标题与层级

| 元素 | 字体 | 字号 | 样式 | 对齐 |
|------|------|------|------|------|
| 论文标题 | Times New Roman | 14pt | 正常 | 居中 |
| 作者 | Times New Roman | 10pt | 正常 | 居中 |
| 一级标题 | Times New Roman | 12pt | 加粗，首字母大写 (`1 Introduction`) | 左对齐 |
| 二级标题 | Times New Roman | 10pt | 加粗，首字母大写 (`1.1 Background`) | 左对齐 |
| 三级标题 | Times New Roman | 10pt | 加粗，非独立行 | 左对齐 |

## 正文

| 参数 | 值 |
|------|-----|
| 字体 | Times New Roman |
| 字号 | 10pt |
| 对齐 | 两端对齐 |
| 首行缩进 | 0.25 inch |
| 行距 | 单倍行距 |
| 段前/段后间距| 0 |

## 摘要

| 参数 | 值 |
|------|-----|
| 字号 | 10pt |
| 样式 | 正常 |
| 缩进 | 左右不缩进 |
| 开头 | 使用 "Abstract" 作为无编号二级标题，居中 |

## 公式

| 参数 | 值 |
|------|-----|
| 格式 | 原生 OMML |
| 位置 | 居中 |
| 编号 | 靠右对齐，格式 `(1)` |

## 图注

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 位置 | 图下方，居中 |
| 前缀 | "Figure N:" 正常 |

## 表注

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 位置 | 表上方，居中 |
| 前缀 | "Table N:" |

## 参考文献

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 缩进 | 悬挂缩进 0.25 inch |
| 编号 | `[1]`, `[2]`, ... |
