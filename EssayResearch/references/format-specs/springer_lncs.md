# Springer LNCS 排版格式规范

> 适用于 Springer 计算机科学讲义（Lecture Notes in Computer Science）等学术专著和会议记录。

## 页面设置

| 参数 | 值 |
|------|-----|
| 纸张 | A4 (210mm × 297mm) |
| 上下左右边距| 页面内容区固定为 122mm × 193mm |
| 分栏 | 单栏 |

## 标题与层级

| 元素 | 字体 | 字号 | 样式 | 对齐 |
|------|------|------|------|------|
| 论文标题 | Times New Roman | 14pt | 加粗 | 居中 |
| 作者 | Times New Roman | 10pt | 正常 | 居中 |
| 机构/邮箱 | Times New Roman | 9pt | 正常 | 居中 |
| 一级标题 | Times New Roman | 12pt | 加粗 (`1 Introduction`) | 左对齐 |
| 二级标题 | Times New Roman | 10pt | 加粗 (`1.1 Background`) | 左对齐 |
| 三级标题 | Times New Roman | 10pt | 加粗，非独立行（Run-in heading） | 左对齐 |
| 四级标题 | Times New Roman | 10pt | 斜体，非独立行 | 左对齐 |

## 正文

| 参数 | 值 |
|------|-----|
| 字体 | Times New Roman |
| 字号 | 10pt |
| 对齐 | 两端对齐 |
| 首行缩进 | 0.5 cm |
| 行距 | 单倍行距 |
| 段前段后间距| 0 |

## 摘要

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 样式 | 正常 |
| 缩进 | 左右各缩进 1 cm |
| 开头 | "Abstract." 加粗，后接正文同行 |

## 公式

| 参数 | 值 |
|------|-----|
| 格式 | 原生 OMML |
| 位置 | 居中 |
| 编号 | 靠右对齐，格式 `(1)` |

## 图注

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 位置 | 图下方，居中 |
| 前缀 | "Fig. N." 加粗或正常 |

## 表注

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 位置 | 表上方，居中 |
| 前缀 | "Table N." 加粗或正常 |

## 参考文献

| 参数 | 值 |
|------|-----|
| 字号 | 9pt |
| 缩进 | 悬挂缩进 0.5 cm |
| 编号 | `[1]`, `[2]`, ... |
